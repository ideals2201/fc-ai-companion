# V0.1.0 - 第一个验收版本

日期：2026-06-06

## 验收结论

项目主人确认：

- 模拟器运行正常。
- 手动操作输入正常。

## 已包含

- 浏览器驾驶舱应用。
- JSNES 真实 ROM 冒烟运行时。
- 本地用户自有 ROM 加载端点 `/api/local-test-rom`。
- Canvas 画面渲染。
- 1P 手动输入：方向、A、B、Start、Select。
- 运行、暂停、重置、开始键脉冲、输入冒烟测试。
- 中文界面。
- Web Audio 声音入口。
- ROM 合规边界：不提交 ROM、不提交真实 ROM 路径。

## 不包含

- RAM state schema。
- CameraX / PlayerX / WorldX 推导。
- Danger Detector。
- Route Script。
- Action Lock。
- FSM。
- AI 伙伴实际战术控制。
- TAS 直接控制。

## 版本性质

这是第一个可运行验收版本，不是最终 MVP。

下一阶段进入战术调试基础层：RAM state schema、WorldX、V0 战术调试驾驶舱面板。
