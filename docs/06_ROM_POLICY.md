# ROM And Asset Policy

This repository must not contain ROMs, BIOS files, save states, copyrighted game assets, or downloaded commercial game archives.

Allowed:

- source code
- documentation
- RAM maps
- derived state schemas
- route landmarks
- tests using legal homebrew/demo ROMs
- local-only instructions for user-owned ROM loading

Forbidden:

- committed `.nes`, `.fds`, `.zip`, `.7z`, `.rar`, save state, or BIOS files
- ROM download URLs
- bundled commercial assets

Local user-owned files belong outside the repository or under ignored local-only folders.
