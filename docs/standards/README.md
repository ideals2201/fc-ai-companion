# FC AI Standards Index

This directory stores versioned standard releases.

## Current Release

- Current stable version: `v1.1.0`
- Directory: `docs/standards/v1.1.0/`

The root-level files under `docs/`, such as `docs/STRATEGY_PACK_STANDARD.md`, remain working aliases for existing project links and tests. Versioned files under this directory are the release snapshots that can be shared with external experts.

## Versioning Rules

- Every released standard document must include the version number in its filename.
- Documents that belong to the same standard release must live in the same version directory.
- A released version directory should be treated as immutable. If the standard changes, create a new version directory, for example `v1.1.0` or `v2.0.0`.
- Draft or research references must not be mixed into a stable release directory unless they are explicitly promoted to that release.
