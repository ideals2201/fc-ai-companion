# Community Communication / 社区沟通准则

## Core Attitude / 基本态度

Everyone here is a learner. The project welcomes people who are curious, patient, and willing to improve together.  
来到这里的每个人都是学习者。项目欢迎保持好奇、愿意耐心交流、愿意共同进步的人。

Technical background is useful, but it is not the only source of value. Clear questions, careful observations, play experience, documentation, testing, and product intuition are also important contributions.  
技术背景很有用，但它不是唯一的价值来源。清楚的问题、细致的观察、游玩体验、文档、测试和产品直觉，同样是重要贡献。

## How We Communicate / 我们如何沟通

- Be respectful and specific. / 保持尊重，并尽量具体。
- Explain what you observed, what you expected, and how to reproduce it. / 说明你观察到了什么、期待什么、如何复现。
- Prefer evidence over status or authority. / 优先看证据，不用身份或权威压人。
- Treat mistakes as useful signals for improving the system. / 把错误视为改进系统的信号。
- Ask when something is unclear. / 不清楚时可以提问。
- Help make the next step easier for the next learner. / 尽量让后来的人更容易继续前进。

## What To Avoid / 避免事项

- Personal attacks, ridicule, or gatekeeping. / 人身攻击、嘲笑或用门槛排斥他人。
- Publishing private information, ROM files, credentials, or local paths. / 公开私人信息、ROM 文件、凭据或本地路径。
- Presenting unvalidated strategy results as confirmed clears. / 把未经验证的策略结果说成已确认通关。
- Hiding uncertainty when evidence is incomplete. / 证据不足时隐藏不确定性。

## Shared Goal / 共同目标

The goal is to learn, build, verify, and improve together.  
共同目标是一起学习、一起构建、一起验证、一起进步。
